# Crypto Algorithmic Trading Research

Research-focused implementation of algorithmic trading strategies
for BTC/USDT and ETH/USDT markets.

The project emphasizes:
- Reproducible data pipelines
- Systematic strategy design
- Backtesting and evaluation

This repository does not include live trading or real-time systems.

> Educational and research use only.

## Acknowledgement

This project was inspired by the problem statement of the
Inter IIT Tech Meet 2023 Quantitative Finance challenge
hosted by Zelta Trading.

The current repository represents an independent,
refactored research implementation and is not a
competition submission.
